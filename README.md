# Цветочный Сад — Telegram Bot

Телеграм-бот с игрой-симулятором цветочного сада. Игроки выращивают цветы разных редкостей, собирают букеты, открывают комбинации с бонусными эффектами, выполняют ежедневные квесты и участвуют в мини-играх.

## Возможности

- **Сад и грядки** — сажайте семена, поливайте растения, собирайте урожай. Если забыть полить — растение засохнет.
- **4 редкости цветов** — обычные (⚪), редкие (🔵), эпические (🟣) и легендарные (🟡) с разным временем роста и доходом.
- **50+ видов цветов** — от розы и тюльпана до монстеры и стефанотиса.
- **Букеты и комбинации** — собирайте букеты из цветов инвентаря. Определённые сочетания активируют эффекты: ускорение роста, бонус к доходу, защита от высыхания и другие.
- **30+ комбинаций** — от «Весенней нежности» (+15% к росту) до «Денежного дерева» (+500 монет в день).
- **Рецепты** — открывайте рецепты комбинаций как награды за квесты и мини-игры.
- **Ежедневные квесты** — задания на посадку, сбор, полив, покупку семян и заработок монет.
- **Мини-игры** — викторина о цветах и угадай число (1–100 за 7 попыток).
- **Аукцион** — торговля цветами между игроками.
- **Автоначисление дохода** — фоновый поток каждую минуту проверяет сад и начисляет монеты за выросшие цветы.

## Структура проекта

```
├── main.py            # Точка входа, запуск бота
├── bot_instance.py    # Экземпляр TeleBot
├── config.py          # Токен, пути к файлам, настройки времени роста
├── handlers.py        # Все хендлеры команд и callback-кнопок
├── game_data.py       # Данные о семенах, цветах, комбинациях, ценах
├── effects.py         # Логика комбинаций и бонусных эффектов
├── services.py        # Фоновый наблюдатель роста, начисление дохода
├── storage.py         # Загрузка/сохранение данных игроков (JSON)
├── quests.py          # Ежедневные квесты и награды
├── minigames.py       # Мини-игры (викторина, угадай число)
├── utils.py           # Вспомогательные функции, логирование ошибок
└── .gitignore
```

## Технологии

- **Python 3**
- **pyTelegramBotAPI** (telebot) — взаимодействие с Telegram Bot API
- **JSON** — хранение данных игроков

## Хостинг

Бот развёрнут на VPS-сервере:

- **IP:** `85.239.37.110`
- **ОС:** Linux (Ubuntu)
- **Управление процессом:** systemd (`flower-bot.service`)
- **Автоперезапуск:** при падении бот автоматически перезапускается через 5 секунд
- **Автозапуск:** бот стартует при перезагрузке сервера

### Управление ботом на сервере

```bash
# Подключение к серверу
ssh root@85.239.37.110

# Статус бота
systemctl status flower-bot

# Перезапуск
systemctl restart flower-bot

# Остановка
systemctl stop flower-bot

# Логи в реальном времени
journalctl -u flower-bot -f

# Обновление кода и перезапуск
cd /root/flower_bot_krulova && git pull && systemctl restart flower-bot
```

### Первоначальная установка на сервер

```bash
ssh root@85.239.37.110

apt update && apt install -y python3 python3-pip python3-venv git

git clone https://github.com/pKa1/flower_bot_krulova.git
cd flower_bot_krulova

python3 -m venv venv
source venv/bin/activate
pip install pytelegrambotapi
deactivate

# Создать systemd-сервис /etc/systemd/system/flower-bot.service
systemctl daemon-reload
systemctl enable flower-bot
systemctl start flower-bot
```
